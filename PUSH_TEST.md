# Push test

This file exists only to test the GitHub Desktop → GitHub → Vercel flow.

If you can see this file at https://github.com/Vettahead/sgas after pushing,
then the connection works and future deploys are just Commit → Push.

Steps to test:
1. Open GitHub Desktop — this file (PUSH_TEST.md) should appear under "Changes".
2. Type a summary like "push test" at the bottom-left.
3. Click "Commit to main".
4. Click "Push origin" at the top.
5. Refresh the GitHub repo page — this file should appear there.
6. (Optional) Check Vercel — a new deployment should start automatically.

Once confirmed, you can delete this file (delete it in File Explorer, then
Commit + Push again, or just leave it — it does nothing).
